
## 快速开始
### 获取源码

代码托管在`NSFI`组下的`blog`中

``` bash
$ git clone https://github.com/NSFI/blog.git
```
### 安装依赖

``` bash
$ npm install
```
### 创建文章
``` bash
$ hexo new '文章title'
```
### 本地服务预览
``` bash
$ hexo server
```
### 生成静态文件
``` bash
$ hexo g
```

More info: [hexo文档](https://hexo.io/zh-cn/docs/index.html)
